In Angular 22, OnPush is the default. ChangeDetectionStrategy.Default has been renamed to ChangeDetectionStrategy.Eager for code that still needs unconditional checking. Below: a "live orders" counter shown three ways.

What changed?

  • OnPush is the default - no need to declare it explicitly.
  • ChangeDetectionStrategy.Default → ChangeDetectionStrategy.Eager.
  • Signals + OnPush is the recommended path; Eager is the escape hatch for mutation code.
