injectAsync() lazy-loads a service behind a dynamic import(), keeping heavy dependencies out of the initial bundle. Both demos below consume the product catalog from the previous chapter (same ProductStoreService singleton).

injectAsync() - PdfGeneratorService (lazy + prefetch: onIdle)

With prefetch: onIdle, the module is warmed up during browser idle time, so the first click is already instant.

@if (lastInvoice()) { 
{{ lastInvoice() }}
}

injectAsync() - CsvExporterService (lazy, no prefetch)

Without prefetch, nothing loads until you click: the first click triggers the dynamic import(), then subsequent clicks reuse the resolved instance.

@if (lastCsv()) { 
{{ lastCsv() }}
}
