3. Custom wait function

Instead of a fixed number of milliseconds, debounced() accepts a function returning a Promise<void> - useful for adaptive delays or skipping the wait after an error. 

debouncedQuery = debounced(this.query, (value, lastSnapshot) => {
  if (lastSnapshot.status === 'error') return;
  const ms = value.length < 3 ? 500 : 200;
  return new Promise<void>((resolve) => setTimeout(resolve, ms));
});
